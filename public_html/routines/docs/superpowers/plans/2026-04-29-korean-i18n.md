# routines UI 한국어화 구현 계획

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** routines.soritune.com의 사용자 메인 화면, 학습 컴포넌트, 시스템 메시지, 어드민 placeholder를 한국어로 전환한다(브랜드성 영어 `Routines`/`Daily English Routine`은 일부 유지).

**Architecture:** i18n 라이브러리 미도입. 반복 라벨과 시스템 메시지는 새 모듈 `src/lib/labels.ts`에 모으고 컴포넌트에서 import. 페이지 고유 헤더 문구는 컴포넌트에 인라인 한국어로 작성. 단일 PR.

**Tech Stack:** Next.js (App Router) + React + TypeScript + pnpm + PM2 + vitest(`src/**/*.test.ts`만 포함하므로 컴포넌트 텍스트 회귀는 수동 grep + 페이지 점검으로 검증).

**Spec:** `docs/superpowers/specs/2026-04-29-korean-i18n-design.md`
